# Manga Downloader 2.0.3

## English

This update focuses on stability and a smoother experience when downloading large manga volumes.

### What's new
- Improved reliability for very large volumes and chapters.
- Large final EPUB files are now automatically split into smaller, reader-friendly parts when needed.
- Better end-of-process dialog: you can now either open the manga folder or simply close the dialog.
- More consistent download and conversion behavior across supported services.
- Improved cleanup flow after volume processing to keep temporary artifacts under control.

### Packaging and distribution
- Updated installer package for macOS (`.pkg`) with valid Developer ID signature.
- Package is notarized by Apple and stapled for smoother installation on macOS.

### Packaging for this release
- Version: `2.0.3`
- Installer package: `Manga-Downloader-macOS-2.0.3.pkg`
