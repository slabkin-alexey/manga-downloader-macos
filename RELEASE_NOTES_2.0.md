# Manga Downloader 2.0

## English

Manga Downloader 2.0 brings a much better first-run experience, clearer update flow, and a smoother onboarding journey for both supported services.

### What's new

- New guided onboarding for each service.
- Separate dedicated tutorials for `Manga.in.ua` and `Com-X Life`.
- Multi-page tutorial flow with clear visual explanations:
  - welcome screen
  - service overview
  - input and setup guide
  - buttons and toggle guide
  - progress tracking guide
- Tutorials now appear only on the first launch of each specific service.

### Better update experience

- Reworked update UI with friendly focused update windows.
- Clear "update available" screen with version context.
- Clear "you are up to date" screen after manual checks.
- Improved messaging and easier actions for users.

### Improved clarity and wording

- More user-friendly text throughout onboarding and update flows.
- Better consistency in explanations and labels across services.
- Expanded localized strings for supported languages.
