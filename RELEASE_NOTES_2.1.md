# Manga Downloader 2.1

## English

This update focuses on better output quality/size balance and a smoother install experience.

### What's new
- Improved image optimization pipeline for more stable results.
- Smarter JPG handling: pages are optimized in a way that keeps visual quality while reducing unnecessary file size growth.
- Better conversion behavior: if a converted image would become larger, the app now keeps the better original result.
- Cleaner files: extra embedded image metadata is removed to help keep output files lean.

### Installer
- New macOS installer package is signed and notarized for safer, cleaner installation on macOS.

### Packaging for this release
- Version: `2.1`
- Installer package: `Manga-Downloader-macOS-2.1.pkg`
