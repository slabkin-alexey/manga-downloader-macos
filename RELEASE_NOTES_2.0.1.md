# Manga Downloader 2.0.1

## English

Manga Downloader 2.0.1 focuses on a smoother everyday experience and better completion flow.

### What's new

- Improved app responsiveness during long-running tasks.
- Smoother live log updates while downloads and conversions are running.
- Better overall stability of progress/output rendering during heavy workloads.
- Updated completion behavior:
  - when a download is finished, opening the folder now takes you directly into your selected save folder (for example, directly into Downloads)
  - this makes post-download access faster and clearer
- General quality and reliability improvements for a cleaner user experience.

### Packaging for this release

- Version: `2.0.1`
- Installer package: `Manga-Downloader-macOS-2.0.1.pkg`
